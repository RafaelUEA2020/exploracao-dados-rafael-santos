# exploracao-dados-rafael-santos
Este é um repositório que comporta a atividade solicitada no classroom da disciplina de Ciência de Dados. Objetivo apenas educacional.

# O que foi feito:
- Tratamento dos dados oriundos do Portal de Dados Aberto, disponibilizado no link: https://dados.gov.br/dados/conjuntos-dados
- Tratamentos realizados: Organização dos dados, filtragem, plotagem de gráficos e cálculo de estatísticas, como Média, moda e mediana. Tudo feito utilizando python, através da ferramenta Google Colab, onde o arquivo está disponível no repositório para livre exploração.
- Redução do tamanho do CSV, visto que ele possuia quase 77000 linhas e 30 colunas, totalizando quase 2310000 de dados, o que estava impossibilitando o Google Colab de processar essa quantidade de arquivos. então ele foi reduzido para 380 linhas, e mantendo o numero de colunas.
- É necessário realizar o upload do CSV disponibilizado no prórpio git para que os códigos no Google Colab funcione.
- Realize o upload no seguinte caminho "/content"
